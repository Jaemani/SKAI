자연어로 물어보세요. 문장마다 클릭 가능한 근거가 붙습니다. 아래 예시를 누르거나 직접 질문하세요.
SKAI — 공중 ISR 융합 코파일럿
여러 공개 소스(항적·위성궤도·기상·뉴스/OSINT)를 온톨로지로 융합해, "이 지역, 지금 하늘에서 뭐가 이상한가?"에 출처가 달린 답을 줍니다. 아래 번호는 화면의 각 구역과 같습니다.
화면 구성
1지도(중앙) — 항공기·위성 지상궤적·이상징후 위치. 마커에 마우스를 올리면 상세.
2이상징후 타임라인(좌) — 탐지된 이상 신호. 클릭하면 지도가 이동하고 근거 관측이 연결선으로 표시되며 승인/기각할 수 있습니다.
3소스·기상·뉴스(우) — 융합된 원천 데이터와 개수. 접기/펼치기 가능.
4코파일럿(하) — 자연어로 물으면 문장마다 근거(cites) 배지를 달아 답합니다. 근거 없는 문장은 시스템이 거부합니다.
핵심 동선 3가지
①질문 → 근거 답변: 아래 입력창에 질문 → 답 문장의 근거 배지 클릭 → 지도에서 그 객체가 강조됩니다.
②이상징후 결정 루프: 좌측 타임라인 클릭 → 근거 확인 → 승인(confirm)/기각(dismiss). Q&A가 아니라 결정으로 이어집니다.
③은닉 정황: "은닉 정황 있어?" 질의 → 서브그래프로 교차소스(위성·뉴스·다른 이상징후) 연관을 그래프로 확인.
정직 안내: 이상탐지는 규칙 기반, 서술은 템플릿(옵션 LLM 다듬기)입니다. 강점은 근거 강제(citation) — 문장은 온톨로지 사실에서 조립되고, 근거 없는 문장은 거부됩니다. 평가 정밀도 수치는 자작 합성 시나리오 기준입니다(라이브 실측 아님).

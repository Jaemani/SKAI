# SKAI — Air ISR Fusion Copilot

> **"이 지역, 지금 하늘에서 뭐가 이상한가?"** — 자연어로 물으면 공개 항적·위성궤도·기상·뉴스/OSINT를 **온톨로지로 융합**해, **모든 문장에 출처가 달린** 상황 요약과 이상징후를 지도·타임라인으로 답하고, 분석가의 승인(Action)으로 결정을 잇는 Maven형 공중 ISR 코파일럿.

## 1. 문제 (Problem Fit)

공중·우주 상황 인식은 항공기 항적, 위성 궤도, 기상, 뉴스/OSINT가 서로 다른 창에 흩어져 있어 분석가가 눈으로 교차하느라 느리고 놓칩니다. 병목은 데이터 양이 아니라 **연결과 의미 도출**입니다. 그리고 AI가 요약을 대신해도 **근거 없는 주장(환각)** 은 군·정보 현장에서 쓸 수 없습니다. SKAI는 이 두 문제를 동시에 겨냥합니다.

## 2. 핵심 설계 — 온톨로지가 척추

- **도메인 온톨로지**: Aircraft·Observation·Track·Satellite·OrbitPass·Region·WeatherState·NewsEvent·Operator·Anomaly·SituationAssessment **11객체·11링크·액션**으로 모델링. "ADS-B가 끊긴 기체 × 같은 시공간의 위성 통과 × 같은 지역을 언급한 뉴스"는 flat table 한 행으로 표현할 수 없는 **그래프 추론**입니다(`correlated_with` 링크, 사유·강도 영속).
- **Citation 강제(환각 구조 차단)**: 답변 문장은 LLM이 생성하지 않습니다. 온톨로지에서 사실을 먼저 확정하고 **문장마다 근거 객체 id(cites)를 물고 태어나며**, 근거 없는 문장은 저장 단계에서 거부됩니다.
- **결정 루프(Q&A가 아님)**: 이상징후 confirm/dismiss가 온톨로지 상태를 영속 전이(human-on-the-loop). 질의 하나가 SituationAssessment 인텔 객체로 남습니다.

## 4. Palantir Foundry / AIP — 구성요소별 정직 명시

- **실사용**: 온톨로지 11객체·**36액션**을 실제 Foundry에 구축, 라이브 왕복 검증. **evidence 없는 이상징후 생성은 Palantir 액션 검증이 거부** — 앱을 우회해 접근해도요. 규칙이 앱이 아니라 **데이터 계층에 사는** 배포 패러다임.
- **AIP Logic 2종 실배선**: `explain-anomaly`(설명·신뢰도)·`region-situation-summary`(상황 헤드라인) — 온톨로지 **객체 참조**를 받아 실제 속성을 읽고 서술 생성(OSDK 타입드 쿼리).
- **Staged Human Review**: AI 산출은 pending → 분석가 승인 시에만 온톨로지 반영(플랫폼 레벨 검증). **Automation**: 신규 비상 스쿽 발생 시 Foundry가 스스로 감지해 알림(실수신 검증).
- **하지 않는 것(정직)**: 탐지·상관·citation 조립은 자체 엔진입니다. AIP는 온톨로지·결정 게이트·서술 생성을 담당 — 이 역할 분담이 로컬 스택으로 재현 못 하는 실질 이점입니다.

## 5. 숫자 — 단, 정직한 숫자

- 라이브 실데이터 질의: 산출 문장 **24/24 전부 출처 인용(cite 해상 100%)**, 인용 id 전건 온톨로지 역추적.
- 같은 데이터를 맨몸 LLM에 주면 기계검증 가능한 출처 **0** vs 파이프라인 **10** (10:0). 차이의 본질은 정확도 경쟁이 아니라 **provenance**.
- 합성 12시나리오 회귀 전건 통과 — 이것은 탐지 로직의 **결정성 검증**이지 실세계 정밀도가 아닙니다(라이브엔 ground truth가 없어 P/R을 원리적으로 낼 수 없음을 명시).
- 테스트 366개 통과. 데모 replay는 **네트워크 0**(소켓 레벨 차단)에서 **바이트 단위 동일 재현**.

## 6. 실행

```bash
scripts/demo.sh replay   # 네트워크 0, 결정적 재현 (합성 시나리오 13건·6유형)
scripts/demo.sh live     # 실데이터 폴링 + 군용 DB플래그·dropout 교차확인·LLM 서술 기본 on
```

합법 가드레일: 공개·무료 API 정상 사용(ToS·크레딧 준수), 표적타격 자동화는 범위 밖. 상세 설계·의사결정·자체 냉정평가는 저장소 `docs/`(DR 13건·EVALUATION.md) 참조.

# DR-0013 — 신호·화면 정직화 라운드 (위성 허용목록 · 군용 서사 · 시간창 · 설명 레이어)

- 날짜: 2026-07-05
- 상태: 채택 (사용자 승인 "진행해")
- 근거: 사용자 실사용 피드백 — "위성 근접 경고가 왜 자꾸 나오는지 모르겠고 의미 없어 보임", "🔗73이 뭔지 설명 없음", "군용기는 오픈소스로 못 얻으면서 합성에선 찾는 척", "코파일럿이 정해진 정보만 나열", "근거·소스 설명이 화면에 안 보임". EVALUATION.md(백엔드 정직화)의 후속 = **신호 품질·화면 정직화**.

## 문제 구조 (2뿌리)

1. **신호 대 잡음**: Celestrak `stations,visual` 카탈로그(ISS·밝은 위성 = ISR 무관) 94기 → KADIZ 통과창 99개/12h → ① 위성 근접 Anomaly 스팸(정상 사건을 경고로), ② 상관 엔진(±60분 버킷)과 곱해져 한 이상징후에 correlated_with 수십 건(실측 73) = 상관의 의미 붕괴.
2. **설명 레이어 부재**: 서브그래프 엣지가 온톨로지 원어 그대로·모달 내 안내 0, Anomaly cite 배지 source_url 상시 공백(`assessment.py:598`), 소스 패널은 카운트만(신선도 미사용), 합성 라벨이 타임라인·설명문에만 전파(지도·요약문장·서브그래프 구분 불가), 🔗N 배지 무설명.
